= check_lib =

==Abstract==
A simple lex&yacc program that checks syntax error of synopsys's .lib format
file.

==How to compile==
  
    $ qmake -project
    $ qmake
    $ make

==How to run==

    $ ./check_lib < power_sample.lib 

==References==
  * ANSI C grammar, Lex specification
